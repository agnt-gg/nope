NOPE

Agent security in one import. Presets to start, fluent builder to customize, one method to scan everything.

Quick Start

npm install @agnt-gg/nope

Works with ESM and CommonJS. Node 18+.

import { NOPE } from '@agnt-gg/nope';

const nope = NOPE.preset('standard');

// Check an action
nope.check({ command: 'DROP TABLE users;' });
// → { allowed: false, violations: [{ rule: 'db-drop-table', severity: 'critical' }] }

// Scan anything — context, MCP tools, plugin code
nope.scan(contextText);
nope.scan({ name: 'shell', description: 'Run commands' });

// Wrap tools — all protection layers active
const safe = nope.wrap(plug.tools);

Presets accept overrides:

const nope = NOPE.preset('standard', { auth: { verifyToken: myFn } });

Fluent builder

Chain .with*() instead of nested config objects:

const nope = new NOPE()
  .withRole('admin', 'critical', 'warn')
  .withRole('agent', 'medium', 'strict')
  .withRateLimit('1m', 60)
  .withSSRF({ enforced: true })
  .withScanners({ homograph: true })
  .withTrust('@agnt-gg/*');

Core API

Four methods cover 90% of use cases.

check(action, context?)

Test an action against all rules. Returns { allowed, violations }.

nope.check({ command: 'DROP TABLE users;' });
// → { allowed: false, violations: [{ rule: 'db-drop-table', severity: 'critical' }] }

nope.check({ tool: 'exec', params: { command: 'ls -la' } });
// → { allowed: true, violations: [] }

// Identity-aware
nope.check({ command: 'shutdown -h now' }, { role: 'admin' });

wrap(tools)

Wrap tools from AI, PLUG, or FLOW. Inputs are checked before execution, outputs are sanitized after. All layers apply automatically.

const safe = nope.wrap(plug.tools);

for (const t of safe) ai.tool(t.name, t.description, t.input, t.run);
// Dangerous inputs → blocked before execution
// Leaked secrets → [REDACTED] before reaching LLM

scan(input)

One method, dispatches by input type:

nope.scan('ignore previous instructions...');       // prompt injection
nope.scan({ name: 'shell', description: '...' });       // MCP tool
nope.scan({ code: pluginSource });                      // plugin code
await nope.scan({ binary: '/usr/bin/tirith' });       // binary checksum

sanitize(value, toolName?)

Scrub secrets from any value. 17 built-in patterns: OpenAI, Anthropic, AWS, Google, GitHub, Groq, Stripe, Replicate, SendGrid, Slack, Twilio, PEM, JWT, Bearer.

const clean = nope.sanitize(toolOutput, 'my-tool');
// sk-ant-secret-key → [REDACTED:Anthropic key]

Configuration

Start with a preset, then layer on what you need. Every feature is opt-in.

Modes & thresholds

new NOPE({ mode: 'strict' })   // block violations (default)
new NOPE({ mode: 'warn' })     // log but allow
new NOPE({ mode: 'audit' })    // silent
new NOPE({ threshold: 'critical' }) // only block critical

Identity & auth

Role-based thresholds, token verification, rate limiting, allow/deny lists, approval memory, and lockout.

const nope = NOPE.preset('standard')
  .withRole('admin', 'critical', 'warn')
  .withRole('agent', 'medium', 'strict')
  .withRateLimit('1m', 60, { admin: 200, agent: 30 })
  .withLockout(5)
  .withAuth({
    verifyToken: async (t) => jwt.verify(t, SECRET),
    allowlist: ['user_abc'],
    denylist: ['user_bad'],
  });

SSRF protection

DNS resolution, redirect chain validation, cloud metadata blocking (AWS, GCP, Azure, Alibaba), IPv6, custom blocklists.

nope.withSSRF({
  enforced: true,          // cannot be disabled
  customBlocklist: ['evil.internal'],
  allowlist: ['safe-api.com'],
});

await nope.resolveAndCheck('https://suspicious.com');
// → { safe: false, reason: 'DNS rebinding: resolves to 10.0.0.1' }

Execution backends

Backends are explicit and fail closed. Docker is the isolation boundary. SSH is remote transport, not a sandbox. host-process requires acknowledgeHostAccess: true and uses an executable plus argument array. WASM shell execution is unavailable and never falls back to the host.

const sb = nope.sandbox({ backend: 'docker', image: 'node:20-slim', pidsLimit: 256, readonlyRoot: true });
await sb.exec('echo hello');
// → { stdout: 'hello\n', exitCode: 0, duration: 42 }

Scanners

Built-in homograph detection, terminal injection scanning, and binary checksum verification. No external binary needed.

nope.withScanners({ homograph: true, terminalInjection: true });

Telemetry & dashboard

nope.withTelemetry({ store: 'memory', retention: '7d' });

nope.report();     // → { totalChecks, blocked, riskScore, topViolations, ... }
nope.dashboard();  // → self-contained HTML dashboard

Smart LLM approval

new NOPE({
  llmApprove: async (cmd, violations) => {
    const r = await llm.chat('Safe? ' + cmd);
    return r.includes('safe') ? 'approve' : 'escalate';
  },
});
LLM can only raise severity, never lower. If it throws, falls through to normal block logic (fail-safe). Verdicts: approve, deny, escalate.

Full Example

NOPE with AI and PLUG — register tools, add guardrails, chat safely.

import { AI } from '@agnt-gg/ai';
import { PLUG } from '@agnt-gg/plug';
import { NOPE } from '@agnt-gg/nope';

// 1. Set up integrations
const plug = new PLUG();
plug.add('shell', {
  description: 'Execute shell commands',
  actions: {
    exec: {
      description: 'Run a command',
      input: { command: 'string' },
      run: async ({ command }) =>
        require('child_process').execSync(command).toString(),
    },
  },
});
plug.connect('shell', {});

// 2. Add guardrails
const nope = NOPE.preset('standard')
  .withRole('agent', 'medium', 'strict')
  .withTrust('@agnt-gg/*');

// 3. Scan context before loading
const ctx = nope.scan(fs.readFileSync('AGENTS.md', 'utf-8'));
if (!ctx.safe) console.warn('Injection detected!', ctx.findings);

// 4. Wrap tools with NOPE protection
const safeTools = nope.wrap(plug.tools);

// 5. Wire into AI
const ai = new AI();
for (const t of safeTools) {
  ai.tool(t.name, t.description, t.input, t.run);
}

// 6. Chat — safe inputs, safe outputs
await ai.chat('List files in current directory');  // works
await ai.chat('Delete everything on the server'); // blocked by NOPE

Advanced

Guard a single function

const safeExec = nope.guard(async ({ command }) => execSync(command).toString());
await safeExec({ command: 'ls' });     // works
await safeExec({ command: 'shutdown -h now' }); // throws

External scanner

Delegate to an external binary. Runs before built-in rules. Verdict is authoritative. Fails open if unavailable.

new NOPE({
  externalScanner: async (action) => ({
    action: (await runBinary(action)).exitCode === 0 ? 'allow' : 'block',
  }),
});

Red team testing

50+ built-in attack vectors with fuzzing. Test your guardrails before attackers do.

const r = await nope.redTeam({ attacks: 'all', iterations: 200 });
// → { passed: 187, failed: 13, coverage: { command: { tested: 48, caught: 45 }, ... } }

Consent callback

new NOPE({
  onBlock: async (violations, action) => {
    return await prompt('Allow? (y/n)') === 'y';
  },
});

Execution pipeline

1. External scanner → 2. Built-in rules → 3. Identity context → 4. Trust level → 5. LLM approval → 6. onBlock callback → 7. Execute tool → 8. Sanitize output
